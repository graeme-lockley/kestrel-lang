# Kanban Epics

Epics group related stories across the roadmap.

## Folders

- `docs/kanban/epics/unplanned/`: active epics
- `docs/kanban/epics/done/`: completed epics

## Naming

- Use `EXX-slug.md`, where `EXX` is the epic identifier.
- Story files should include a `## Epic` section linking to exactly one epic file.

## Completion

Move an epic file from `epics/unplanned/` to `epics/done/` when:

- All member stories are in `docs/kanban/done/`.
- Epic completion criteria in the epic file are satisfied.
- Any deferred work is linked as a follow-up story/epic.
