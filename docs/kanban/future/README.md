# Future (investigations and ideas)

This folder holds **future** kanban items: observations, investigations, and possible directions that are **not** on the prioritized roadmap yet.

They use **`slug.md` filenames only** (no `NN-` prefix). See **[docs/kanban/README.md](../README.md#future-investigations-and-ideas)** for naming, lifecycle, and promotion to **unplanned**.

When an item is ready to become real work, promote it to **`docs/kanban/unplanned/`** as **`NN-slug.md`** with a new global sequence number and full unplanned sections.
