# Future (investigations and ideas)

This folder holds **future** kanban items: observations, investigations, and possible directions that are **not** on the prioritized roadmap yet.

They use **`slug.md` filenames only** (no `S##-##-` prefix). See **[docs/kanban/README.md](../README.md#future-investigations-and-ideas)** for naming, lifecycle, and promotion to **unplanned**.

When an item is ready to become real work, promote it to **`docs/kanban/unplanned/`** as **`S##-##-slug.md`** (epic id + story order within epic) and fill full unplanned sections.
